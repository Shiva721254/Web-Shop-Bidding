# Web Shop Bidding

A full-stack auction starter with a PHP REST API and a Vue 3 frontend.

## Project Layout

- [backend/README.md](backend/README.md) — backend setup, API endpoints, and container notes.
- [frontend/README.md](frontend/README.md) — frontend setup, scripts, and component guidance.

## Quick Start

```bash
cd backend
docker compose up
docker compose exec php composer install
```

```bash
cd frontend
npm install
npm run dev
```

The backend runs at `http://localhost`, and the frontend runs at the Vite URL, usually `http://localhost:5173`.
